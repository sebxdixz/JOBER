# Code of Conduct

Este proyecto adopta el [Contributor Covenant](https://www.contributor-covenant.org/) version 2.1.

## Nuestro compromiso

Nos comprometemos a ofrecer una experiencia libre de acoso para todos, sin importar edad, origen, identidad o nivel tecnico.

## Comportamientos esperados

- Respeto y colaboracion.
- Feedback tecnico claro y directo.
- Cuidar la privacidad de otros.

## Comportamientos inaceptables

- Ataques personales, discriminacion o acoso.
- Publicar datos sensibles de terceros.

## Alcance

Este codigo aplica a todos los espacios del proyecto.

## Enforcement

Si necesitas reportar un problema, abre un issue o contacta a los mantenedores.
